# 🚀 AI Digital Twin: Business Simulation & Decision Intelligence System

An advanced AI-powered system that simulates business scenarios and predicts outcomes using a combination of Machine Learning, Multi-Agent Architecture, and Retrieval-Augmented Generation (RAG).

This project creates a **digital twin of a business**, enabling users to test different strategies and receive data-driven recommendations.

---

## 🧠 Key Features

- Multi-Agent AI System (Planner, Research, Simulation, Analysis, Decision)
- RAG-based knowledge retrieval using FAISS
- Business scenario simulation using Machine Learning
- Decision intelligence engine
- Memory system for storing past insights
- Interactive dashboard using Streamlit

---

## 🏗️ Architecture

User Input → Planner Agent → Research Agent (RAG) → Simulation Agent (ML Model) → Analysis Agent → Decision Agent → Memory → Dashboard Output

---

## 🛠️ Tech Stack

- Python  
- Scikit-learn (ML Models)  
- SentenceTransformers (Embeddings)  
- FAISS (Vector Database)  
- Streamlit (Dashboard UI)  
- Pandas / NumPy
